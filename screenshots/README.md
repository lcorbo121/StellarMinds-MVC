# Capturas de pantalla

Colocá aquí tus capturas con estos nombres exactos para que aparezcan en el README principal:

| Archivo | Pantalla sugerida |
|---------|-------------------|
| `login.png` | Pantalla de inicio de sesión (Usuarios/Index) |
| `equipos.png` | Listado de equipos (telescopios, cámaras, etc.) |
| `observaciones.png` | Observaciones o el ranking |
| `prestamos.png` | Préstamos / devolución / auditoría |

Formato recomendado: **PNG**, ancho ~1200–1600px. Podés agregar más y referenciarlas en el README.
